Простая аркадная игра, написанная на Python с использованием библиотеки PyGame.
Цель игры — сбивать падающие цели и защищать нижнюю границу экрана.

Как запустить:
git clone https://github.com/flikzstyle/PyGame.git
cd PyGame

для WSL/Linux/MacOs: 

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python3 main.py

Управление
Стрелки ВЛЕВО / ВПРАВО — gеремещение пушки
ПРОБЕЛ (Space) — cтрельба
ENTER — подтверждение ввода имени
